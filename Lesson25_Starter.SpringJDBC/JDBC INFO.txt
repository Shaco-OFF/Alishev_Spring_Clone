 Реляционные БД - БД в которых данные хранятся в виде таблиц
 Столбцы БД - это поля объекта
 Строки БД - это сами объекты с конкретными значениями полей

▶▶  SQL-команды:

▶DDL-команды (Data Definition Language): - набор SQL-команд для создания схем, сущностей

    ▶CREATE DATABASE db_name; - создание базы данных с указанным именем
    ▶DROP DATABASE db_name; - удаление базы данных с указанным именем
    ▶CREATE TABLE table_name ( - создание таблицы с указанными колонками (которые соответствуют полям класса)
        id int,
        name varchar
    );
    ▶DROP TABLE table_name; - удаление таблицы с указанным именем

▶DML-команды (Data Manipulation Language) - набор SQL-команд для управления данными

    ▶ Получение набора строк (значений) для указанных столбцов (полей)
    SELECT id, name FROM db_name;
    ▶Получение набора строк (значений) для ВСЕХ столбцов (полей)
    SELECT * FROM db_name;
    ▶Добавление строк с указанными значениями полей
    INSERT INTO Person(id, name) VALUES (1, 'Tom');
    ▶Изменение указанных строк (значений) по одному из полей (после команды WHERE)
    UPDATE Person SET name='Tom123' WHERE id=1; //в данном случае изменяется имя человека с указанным id
    ▶Удаление строки из таблицы по одному из полей (после команды WHERE)
    DELETE FROM Person WHERE id=1; //в данном случае удаляется человек с указанным id
